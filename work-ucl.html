STIGMERGIC SPACES / UCL BARTLETT

极端环境下的
无人建造机器人的原型开发

面向极端环境与无人参与的未来建造场景，设计基于 Unity3D 的多机器人建造仿真系统，结合路径规划、强化学习与力反馈算法，实现机器人对模块化构件的自主搬运、协作搭建与空间重构。

Unity3D ML-Agents PPO Robot Simulation Spatial Intelligence Autonomous Construction
System Logic

材料、机器人与算法共同反馈的无人建造系统。

项目面向极端环境或无人参与的未来建筑搭建任务，借鉴蚂蚁、白蚁、黏菌等自然群体的自组织行为，将建筑空间理解为由多个智能体共同协商、反馈和重构的动态系统。设计了一套基于 Unity3D 的多机器人建造仿真系统，并通过强化学习与力反馈算法，使机器人能够在复杂环境中完成自主路径规划、构件搬运、姿态调整与模块搭建。

01
Material System

模块化构件系统

构件采用木质模块与斜角节点设计，支持连接、拆卸、重组和多方向聚合，为机器人操作提供清晰的几何接口。

02
Robotic System

多机器人协作

小型机器人通过移动底盘、旋转关节和夹持机构完成构件搬运、姿态调整、桥接与辅助搭建。

03
Computational System

仿真与学习控制

基于 Unity3D、C# 与 ML-Agents 构建训练环境，通过 PPO 强化学习训练机器人完成目标导向行为。

04
Feedback System

力反馈与结构稳定性

系统读取构件位置、压力、扭矩与受力数据，并通过图结构可视化和机器学习辅助判断结构是否稳定。

01 / Robotic Prototype

通过 Unity 仿真与物理原型验证机器人动作

机器人系统围绕模块化构件的搬运、旋转、传递和放置进行设计。原型经历多轮迭代，逐步整合滑轨移动、旋转关节、夹持机构与双机器人协作能力。测试内容包括 Y 轴旋转、Z 轴旋转、X 轴滑动、构件传递、双机器人协同放置以及辅助汇合等动作。

项目在 Unity 中建立虚拟控制环境，并通过 Raspberry Pi、舵机与电源模块连接实体机器人，实现仿真动作与物理原型之间的对应测试。

02 / Collaboration Simulation

水平与垂直方向的构件协作

在机器人协作仿真中，系统测试了水平 Pivot、垂直 Pivot、同层旋转、支点切换与穿越几何等协作动作。理解机器人之间如何共享构件重量、改变支点、交换操作角色，并在有限空间中完成构件的方向转换。

Unity 仿真环境中，通过路径规划对构件的自主搬运、旋转和放置动作进行模拟

通过 Raspberry Pi 控制物理原型进行对应测试。

协作仿真重点验证机器人之间如何分担构件重量、调整支点位置，并在有限空间中完成构件旋转和传递。

03 / Reinforcement Learning

通过RL训练机器人完成目标导向的自主控制

项目使用 Unity ML-Agents 构建训练环境，并结合 PPO 强化学习方法训练机器人行为。训练任务包括识别目标位置、调整自身姿态、移动到指定区域、抓取目标物并将其运输到目标点。在训练过程中，机器人状态包含自身位置、目标位置、构件相对方向、关节状态与检测器反馈；动作空间对应多个旋转关节、滑动方向与夹持行为；奖励机制则根据接近目标、正确抓取、成功放置和避免碰撞等条件进行设计。

State

状态定义

包括机器人位置、目标点、构件方向、关节状态、检测范围与环境约束。

Action

动作控制

控制机器人旋转、滑动、抓取、释放以及多个关节的姿态变化。

Reward

奖励机制

根据接近目标、正确抓取、成功运输、减少碰撞和完成任务给予正负反馈。

Training

PPO 训练

使用 ML-Agents PPO 方法，使机器人在大量仿真试错中学习稳定的任务完成策略。

04 / Spatial Rearrangement

从机器人建造动作转化为空间装配、拆卸与重构场景。

项目最终将机器人控制与算法系统转化为建筑空间中的装配和重构场景。通过优化建造序列，系统可以根据不同地形条件调整构件放置路径，使结构适应不平整地面、坡地或悬崖等极端环境。 在室内场景中，机器人可参与墙体单元的移除与安装，使空间布局根据使用需求进行动态调整。个人居住空间、集体住宅和空间原型共同展示了该系统在未来 living architecture 中的潜在应用。
